# Images Directory

Place your visualization images in this directory with the following filenames:

## Data Exploration Images
- `churn_distribution.png` - Distribution of Customer Churn (bar chart)
- `tenure_distribution.png` - Distribution of Tenure histogram
- `monthly_charges_distribution.png` - Distribution of Monthly Charges histogram
- `total_charges_distribution.png` - Distribution of Total Charges histogram
- `churn_by_contract.png` - Churn Rate by Contract Type (bar chart)
- `tenure_vs_charges.png` - Tenure vs. Monthly Charges scatter plot
- `monthly_charges_by_churn.png` - Monthly Charges by Churn Status box plot

## Analysis Images
- `correlation_matrix.png` - Correlation Matrix heatmap
- `feature_importance.png` - Top 15 Feature Importances bar chart

## Results Images
- `roc_random_forest.png` - ROC Curve for Random Forest Classifier
- `roc_xgboost.png` - ROC Curve for XGBoost Classifier
- `roc_logistic.png` - ROC Curve for Logistic/Lasso Regression
- `confusion_matrix_random_forest.png` - Confusion Matrix for Random Forest
- `confusion_matrix_xgboost.png` - Confusion Matrix for XGBoost
- `confusion_matrix_logistic.png` - Confusion Matrix for Logistic Regression

## Notes
- Supported formats: PNG, JPG, JPEG
- Images will be automatically displayed in the relevant sections of the app
- If an image is missing, the app will show a helpful message
